📚 Educare LMS

Educare LMS is a full-stack Learning Management System that helps manage online learning in a simple and efficient way. It allows users to access courses, while admins can manage users, content, and the system from a dashboard.

✨ Features

User authentication

Course and learning content management

Admin dashboard

Secure backend APIs

Responsive frontend design

🛠️ Tech Stack

Frontend

React (Vite)

JavaScript

Axios

Backend

Node.js

Express.js

MongoDB (Atlas)

JWT Authentication
